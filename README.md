
# Electricity Usage Analysis ⚡

This project analyzes electricity consumption data using Python.

It detects abnormal electricity usage patterns and predicts future electricity demand.

## Features
- Electricity usage visualization
- Spike / anomaly detection
- Electricity usage prediction
- Energy saving suggestions
- Bill estimation

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Project Goal
The goal of this project is to analyze electricity usage patterns and provide insights into energy consumption behavior.
